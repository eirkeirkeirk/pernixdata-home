  • Storage and VM Acceleration Platform

    FVP moves storage reads and writes in to the server tier using clustered flash/RAM, ensuring the fastest VM performance with seamless operations, scale-out growth, and complete fault tolerance.

    Any shared storage hardware can be used for capacity, maximizing design flexibility and avoiding rip-and-replace headaches.
